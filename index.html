CNC Machine Control System

Modern control system for CNC machines using Electron and Python.

Project Overview

The CNC Machine Control System is a custom solution designed to replace the Universal G-code Sender (UGS) with a modern, user-friendly application. The system uses Electron for the frontend and Python for the backend, providing advanced features like real-time monitoring, G-code editing, and safety systems.

Key Features

  • Real-time machine status and position updates.
  • G-code editing and management.
  • Safety systems (limit switches, emergency stop, failure detection).
  • User-friendly interface for machine operation.
